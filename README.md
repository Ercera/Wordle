# Wordle
Это приложение представляет собой визуальную копию игры "Wordle". 
Оптимизировано для работы на различных устройствах.

Идея, лежащая в основе этого проекта, заключалась в тестировании проекта на reactJS и сборке проекта.
В основе игры заложен поиск слова из 2тыс. возможных вариантов и ограничение в 5 попыток на поиск нужного слова.

### Решенные проблемы
- Решена проблема с выгрузкой слов из словаря, при перезагрузке страницы.
- Проверка вводимых слов отрабатывает корректно, помогая игроку с поиском слова.

### Известные проблемы
- Данный проект не является легко расширяемым. (Если мы захотим улучшить игру добавив регулировку длинны слов, то нужно будет переписывать много кода).
- Есть некоторые хуки, которые необходимо доработать (useCallback/useEffect), выполнению логики они не мешают, но у них отсутствуют зависимости.
